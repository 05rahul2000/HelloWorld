
# Installation 


**How to install Kotlin on Ubuntu**

```
$ sudo apt-get update

Install desired Java version : 

$ sudo apt install openjdk-11-jdk 

$ sudo snap install --classic kotlin
```



# How to run

```
$ kotlinc HelloWorld.kt -include-runtime -d HelloWorld.jar

$ java -jar HelloWorld.jar
```
