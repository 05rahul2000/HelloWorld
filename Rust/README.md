## Installation 

### How to install Rustup on Ubuntu

```
$ sudo apt-get install curl -y

$ curl https://sh.rustup.rs -sSf | sh
```

---

### How to install Rustup on red hat
```
$ sudo dnf install curl -y

$ curl https://sh.rustup.rs -sSf | sh
```

---

### How to run helloworld

```
$ rustc HelloWorld.rs
```
